9.4.1.3: Reading from a File
Files you open for read access (using ios::in) must exist already, or C++
gives you an error message. You can’t read a file that does not exist.
Open() returns zero if the file does not exist when you open it for read
access.
Another event happens when you read files. Eventually, you read all the
data. Subsequently reading produces error because there is no more data to
read. C++ provides a solution to the end-of-file occurrence.
If you attempt to read a file that you have completely read the data from,
C++ returns the value zero. To find the end-of-file condition, be sure to
check for zero when reading information from files.
The following code asks the user for a file name and displays the content
of the file to the screen.
#include<fstream.h>
#include<conio.h>
#include<stdlib.h>
void main( )
{
clrscr()
char name[20],filename[15]
ifstream indata
cout<<"\nEnter the file name : "
cin.getline(filename,15)
indata.open(filename,ios::in)
if(indata.fail())
{
cout<<"\nError opening file : "<<filename
}
while(!indata.eof())// checks for the end-of-file
{
indata>>name
cout<<name<<endl
}
indata.close()
}


Reading characters from a sequential file
You can read a characters from a file using get() function. The following
program asks for a file name and displays each character of the file to the
screen. NB. A space among characters is considered as a character and
hence, the exact replica of the file will be shown in the screen.

#include<fstream.h>
#include<conio.h>
#include<stdlib.h>
void main()
{
char c,filename[15]
if(indata.fail())// check id open succeeded
{
cerr<<"\nError opening file : "<<filename
}
while(!indata.eof())// check for eof
{
indata.get(c)
cout<<c
}
9.4.1.3 ከፋይል ላይ ማንበብ (Reading from a File)
​ከፋይል ላይ መረጃ ስታነብ ማስታወስ ያለብህ ዋና ዋና ነገሮች፦
​ፋይሉ መኖር አለበት፦ ለማንበብ የምትከፍተው ፋይል (ios::in) አስቀድሞ በኮምፒውተርህ ላይ መኖር አለበት። ከሌለ C++ ስህተት (error) ይሰጥሃል።
​የፋይል መጨረሻ (End-of-File - EOF)፦ በፋይል ውስጥ ያለውን ዳታ አንብበህ ስትጨርስ "መጨረሻ ላይ ደርሻለሁ" የሚል ምልክት ያስፈልጋል። ለዚህ ደግሞ eof() የምትባለውን ፋንክሽን እንጠቀማለን።
​ምሳሌ 1፦ ቃላትን/ስሞችን ከፋይል ማንበብ
​ይህ ፕሮግራም ተጠቃሚው የፋይል ስም እንዲያስገባ ይጠይቃል፣ ከዚያም በፋይሉ ውስጥ ያሉትን ስሞች አንድ በአንድ እያነበበ ስክሪን ላይ ያሳያል።
#include<fstream.h>
#include<conio.h>
#include<stdlib.h>

void main()
{
char name[20], filename[15]

cout<<"\nEnter the file name : "
cin.getline(filename, 15)

indata.open(filename, ios::in) // ፋይሉን ለማንበብ መክፈት

if(indata.fail()) // ፋይሉ መኖሩን ማረጋገጥ
{
cout<<"\nError opening file : "<<filename
}

// የፋይሉ መጨረሻ ላይ እስክንደርስ ድረስ እያነበብክ አውጣ (while loop)
while(!indata.eof()) 
{
indata >> name // ከፋይሉ አንብበህ 'name' ውስጥ ክተተው
cout << name << endl // ስክሪን ላይ አሳየው
}

indata.close()
}
ምሳሌ 2፦ ፊደላትን (Characters) አንድ በአንድ ማንበብ
​አንዳንድ ጊዜ በቃላት ፋንታ እያንዳንዱን ፊደል (space ወይም ነጥብን ጨምሮ) ማንበብ እንፈልጋለን። ለዚህ ደግሞ get() የምትባለውን ፋንክሽን እንጠቀማለን። ይህ ፋንክሽን በፋይሉ ውስጥ ያለውን ክፍት ቦታ (space) እንደ አንድ ፊደል ስለሚቆጥረው ፋይሉ ያለውን ይዘት ልክ እንዳለ (exact replica) ያሳየናል።
#include<fstream.h>
#include<conio.h>
#include<stdlib.h>

void main()
{
char c, filename[15]

if(indata.fail())
{
cerr<<"\nError opening file : "<<filename
}

while(!indata.eof())
{
indata.get(c) // አንድ ፊደል ከፋይሉ ማንበብ
cout << c // ያነበብነውን ፊደል ስክሪን ላይ ማሳየት
}

indata.close()
}
በአጭሩ፦
​>> ስንጠቀም ቃላትን (ቃላትን የሚለየው ክፍት ቦታ ወይም space ነው) ያነብልናል።
​get(c) ስንጠቀም ግን ክፍት ቦታዎችንም ጭምር አንድ በአንድ ያነብልናል።

​seekp(): መጻፊያ (put) ፖይንተሩን ወደ ፈለግነው ቦታ ያንቀሳቅሰዋል።
​tellg(): ማንበቢያ ፖይንተሩ አሁን የት ቦታ ላይ እንዳለ ቁጥሩን ይነግረናል።
​tellp(): መጻፊያ ፖይንተሩ ያለበትን ቦታ (index) ይነግረናል።
​ጠቃሚ ማስታወሻ: ለምሳሌ አንድን ፋይል በ ios::app ከፍተህ tellp() ብትል፣ የፋይሉን ጠቅላላ ርዝመት (Size) ይነግርሃል። ምክንያቱም ፖይንተሩ ያለው ፋይሉ መጨረሻ ላይ ስለሆነ።
​የ seekg() አጠቃቀም በምሳሌ
​ይህ ፕሮግራም ከአልፋቤት A እስከ Z ፋይል ውስጥ ይጽፍና፣ በፈለግነው ቁጥር ላይ ያለውን ፊደል እየዘለለ ያነብልናል።
// ... header files ...
void main() {
fstream fileobj
char ch
fileobj.open("d:\\alph.txt", ios::out | ios::in)

for(ch 'A' ch <'Z' ch++) { fileobj << ch } // A-Z ጻፍን

// ከጅምሩ (ios::beg) 8 ቦታዎችን ዝለል
fileobj.seekg(8, ios::beg) 
fileobj >> ch
cout << "\n8ኛው ፊደል : " << ch // 'I' ያወጣል

// ከጅምሩ 16 ቦታዎችን ዝለል
fileobj.seekg(16L, ios::beg)
fileobj >> ch
cout << "\n16ኛው ፊደል : " << ch // 'Q' ያወጣል
}
ፋይልን ከኋላ ወደ ፊት ማንበብ (Reverse Reading)
​አንድን ፋይል ከግርጌ ወደ ላይ ማንበብ ከፈለግክ፣ መጀመሪያ ፖይንተሩን ወደ መጨረሻው መውሰድ አለብህ።
fileobj.seekg(0L, ios::end) ካልህ "ከፋይሉ መጨረሻ 0 ባይት ራቅ" ማለት ነው፣ ይህም ልክ ፋይሉ ማለቂያ ላይ ያቆምሃል።
​ከታች ያለው ፕሮግራም ደግሞ አንድን ፋይል ወደ ኋላ እየተጓዘ እንዲያነብ የተሰራ ነው፦
// ... header files ...
void main() {
ifstream indata
char inchar
indata.open("c:\\alph.txt", ios::in)

// ወደ መጨረሻው ፊደል ሂድ
indata.seekg(-1L, ios::end) 

for(int ctr0 ctr<26 ctr++) {
indata >> inchar // አንድ ፊደል ያነባል (በዚህ ጊዜ ፖይንተሩ ወደ ፊት ይሄዳል)
indata.seekg(-2L, ios::cur) // ስለዚህ ደግሞ 2 ቦታ ወደ ኋላ እንዲመለስ እናዘዋለን
cout << inchar
}
}
