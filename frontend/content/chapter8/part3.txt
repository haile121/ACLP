5.4 Arrays of Structures

Arrays of structures are useful for storing collections of related data sets such as employee records, inventory files, or any data that fits a structure format.

---

 Structure Declaration with Arrays

Consider the following structure definition for a company’s store data:

struct Company
{
    int employees
    int registers
    double sales
} store[1000]
- This single declaration creates an array store of 1000 Company structures.
- Each element in the array contains three members: employees, registers, and sales.
- Note: Creating large arrays of structures consumes memory quickly, so ensure enough system resources are available.


You can also first declare a structure and then define the array separately:

struct Company
{
    int employees // Structure defined without variables

// Later in the program
Company store[1000] // Array of 1000 Company structures
---

 Accessing and Modifying Array of Structures

- Use the dot operator (.) to access members of individual elements:

store[4].employees +3  // Increase employees count of 5th store by 3
- Unlike simple arrays, you can assign one entire structure element to another:

store[44] store[19]  // Copy all members from 20th store to 45th store

}

Referencing the array structure
The dot operator (.) ለ structure array element ልክ ለ normal variables እንደሚሰራው ነው የሚሰራው። ለአምስተኛው store (store[4]) የ employees ቁጥር በሶስት ቢጨምር፣ structure variable እንደዚህ update ማድረግ ትችላለህ፦

store[4].employees +3
ልክ arrays ላይ እንዳየነው፣ የአንድን array ሙሉ content በ simple statement ወደ ሌላ array copy ማድረግ እንደማንችለው፣ በ structures ውስጥ ግን ሙሉ structures በ array notation በመጠቀም assign ማድረግ ትችላለህ።

የ20ኛውን store አባላት በሙሉ ወደ 45ኛው store assign ለማድረግ፣ እንዲህ ታደርጋለህ፦
store[44] store[19]//copies all members from 20th store to 45th
